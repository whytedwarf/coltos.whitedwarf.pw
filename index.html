features

Colt Interiors, an exclusive lounge with numerous customization options for users, help customize many device functions and it's look and feel according to a user's taste and preferences.

Colt Interiors

Statusbar customizations like date, time, logos etc from within the Colt Interiors.

Colt Interiors customizable animations option makes it's page view swipe into animated view.

Quick Settings

Quick Settings customizations with various newly added features like weather, pie control etc along with other functions like Wifi, Data, Bluetooth, Battery, Torch etc.

Blur Personalization

Blur Personalization feature includes Statusbar, Recents & Quick Settings blurring with customizable depth and degrees of translucency.

Pie Controls

Pie Control makes it easier to get the navbar functions on either sides of the device screen, left or right.

Omni Switch

Omni Switch gives a lot of functional features inclusive of Recents, App Drawer, Launcher and other device functions.

Ambient Display

Ambient Display feature helps users check for notifications as the device is taken out of the pocket or picked up in the hand.

Gestures

Gesture Anywhere helps the user via drawn gesture patterns assigned to access a number of device functions included within settings from any screen, like Colt Interiors, Battery, Sound, Bluetooth, Privacy Guard and various other settings.

Touchscreen Gestures helps the user interact with the device even in it's sleep mode by drawing alphabetical letter shapes on the inactive screen, to directly use a function like Torchlight, Email, SMS Text Messaging, Camera etc.

About ColtOS

COLT OS is not only a custom ROM, it's much more than that. First up, it's a family. Then a team. Next it's a project. And finally, it emerges as a custom ROM par excellence.

Colt OS is a custom android ROM based on the Lineage Operating System(LOS). This ROM is a culmination of months of work since December 2016 on the android Nougat platform by a team of dedicated learners of android ROM developers and testers. It contains cherry picked utility features as well as attractive operational convenience features too. Being developed for a number of makes and models of android smartphones, this ROM has become a very popular daily driver for numerous users of those devices instantly at it's initial release of the version 1.0.

Currently the team is working intensively and is just at the threshold of releasing the first Oreo version of Colt OS. Do keep an eye on the community page of the Colt OS project.
